# SQL Fraud Detection System

## Project Overview
Built a fraud detection pipeline using SQL to identify 
suspicious transaction patterns in financial datasets 
by designing a complete database with multiple 
interconnected tables and analytical queries.

## Database Structure
- 5 Tables created (Customers, Transactions, 
  Login Attempts, Accounts, Alerts)
- 20+ records per table for realistic analysis

## Queries Implemented
| Query | Description |
|-------|-------------|
| Query 1 | Multiple Location Transactions Detection |
| Query 2 | Large/Unusual Transaction Flagging |
| Query 3 | Failed Login Attempt Monitoring |
| Query 4 | Blocked Account Identification |
| Query 5 | High Risk Alert Generation |
| Query 6 | Customer Fraud Summary Report |

## Skills Used
SQL, Database Design, JOINs, GROUP BY, 
HAVING, Aggregations, Conditional Logic, 
Pattern Recognition, Fraud Analytics

## Outcome
Successfully identified suspicious transaction patterns 
simulating a real-world bank fraud detection system.
